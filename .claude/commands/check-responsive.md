Vérifie le responsive design du portfolio :
1. Lis les fichiers CSS et identifie toutes les media queries présentes
2. Vérifie que les breakpoints standards sont couverts (mobile < 768px, tablette 768-1024px, desktop > 1024px)
3. Identifie les éléments HTML qui pourraient poser problème en mobile (images sans max-width, tableaux larges, texte avec taille fixe en px)
4. Produis un rapport avec les points OK et les points à corriger

Si $ARGUMENTS est fourni, concentre l'analyse sur ce fichier spécifique.
